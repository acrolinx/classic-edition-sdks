enum AcroSDK::TBreakLevel

Defines available break levels for separating text blocks.

The break level defines how a text block is related to the following block. The text extraction algorithm has to "translate" the application-specific text segmentation (text flow information ) into the SDK text model. The Acrolinx Server is not aware how the submitted text is structured in the originating document. Therefore it is important to choose appropriate break levels for the text blocks.

Enumerator:
BL_NONE 

There is nothing in between this text block and the next block. This option is the default. When the text itself does not contain separating characters like a space, the text is concatenated.

BL_WORD 

The end of this text block can be considered as the end of a word.

BL_SENTENCE 

The end of this text block can be considered as the end of a sentence. Typical example: items of a bulleted list or a paragraph tag.

BL_PARAGRAPH 
BL_TABLE_CELL 

The text block is the last fragment of a table cell. The next text block might be the next cell's content or text behind the table. Nevertheless, the following text block is not grammatically linked to this one.

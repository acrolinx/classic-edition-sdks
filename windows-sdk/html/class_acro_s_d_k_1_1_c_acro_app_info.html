Detailed Description

Application information (name, version...)

This class defines your application details (short name, friendly name, version, build number, revision number, and plug-in signature as provided by Acrolinx). This information is used for storing application-specific configuration in the user profile. The plug-in name and version is listed in the check reports as well for support purposes.

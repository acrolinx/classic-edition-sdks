class CAcroCheckResult2

Detailed Description

The main API class.

This class is the main entry point for checking text. Best practice is to have one single instance of this class in the client application.

virtual bool AcroSDK::CAcroAPI2::addToAggregation ( ) [virtual]

Add checking report of last check to aggregated report.

For a regular batchCheckText / fileSetCheck, the aggregated report is generated automatically. However, it is possible to generate an aggregated report of multiple checked files. After checking a file, call addToAggregation() for adding the checking report to the list of reports to be aggregated. The aggregated report is finally generated by calling commitAggregation(...).

True on success.

virtual int AcroSDK::CAcroAPI2::showModalMessageBox ( HWND  hParent,
const wchar_t *  message,
const wchar_t *  caption,
UINT  uMessageBoxType,
const wchar_t *  checkBoxDontShowId,
bool  showElevationIcon = false 
) [virtual]

Show modal message box.

Show modal message box with given message and caption. When no caption is provided, the SDK uses an appropriate default title. The method support the following Win32 message box styles: MB_OK, MB_OKCANCEL, MB_YESNO, MB_RETRYCANCEL, MB_ABORTRETRYIGNORE, MB_YESNOCANCEL, MB_CANCELTRYCONTINUE, MB_ICONHAND, MB_ICONQUESTION, MB_ICONEXCLAMATION, MB_ICONASTERISK. When checkBoxDontShowId is provided, the messagbox will contain a checkbox "Do not show this message again". When the user checks that checkbox and confirms the dialog with OK / YES, a message box with the same "checkBoxDontShowId" value is not displayed again on that machine for that user. When showElevationIcon is true, the OX / YES button will have the UAC shield icon (elevation).

Returns:
The win32 id of the chosen button: IDOK, IDCANCEL, IDYES, IDNO, IDRETRY, IDABORT, IDIGNORE, IDYES, IDNO, IDTRYAGAIN, IDCONTINUE.
Parameters:
[in]hParentparent window handle
[in]messagemessage to be displayed
[in]captionmessage box title, use empty string or NULL for default
[in]uMessageBoxTypemessage box type
[in]checkBoxDontShowIdOptional. NULL or any message-specific string constant
[in]showElevationIconOptional. Set to true when the OK / YES button should have the windows shield icon (UAC)

Detailed Description

Series of text and info blocks.

The Windows SDK proprietary text model is used for handling text in a generic, application-independent way. It supports attaching additional context information for the linguistic analysis on the Acrolinx Server. Extract the text content from the original document into a series of text and info blocks. When using the corrections dialog, the SDK will return a copy of the submitted CTextDoc that contains all corrections to be written back to the original document.

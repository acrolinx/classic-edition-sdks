Detailed Description

Extracted non-text content.

Objects of class AcroSDK::CInfoBlock represent application-specific embedded objects like graphics. Info blocks are used for telling the server that the text contains objects that cannot be checked. The SDK sends a so-called empty element to the server. The linguistic rules on the server recognize empty elements as valid linguistic content.

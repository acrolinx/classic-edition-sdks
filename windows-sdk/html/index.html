Acrolinx Windows SDK

The Windows SDK allows an easy integration of Acrolinx into third-party applications such as authoring tools, content management systems, or database front ends. The SDK contains methods for text correction. When using the Windows SDK, the developer does not need to handle linguistic error types etc., everything works out-of-the-box. Out-of-the-box, we ensure a unified look and feel, and the user experience is the same for all plug-ins and integrations. User interface elements like the options dialog, progress bar, check result, corrections dialog, or the flag shortcut menu are common to all implementations. New SDK-internal features are automatically available for all plug-ins and integrations. The SDK abstraction layer minimizes the impact of server protocol changes on external integrations.

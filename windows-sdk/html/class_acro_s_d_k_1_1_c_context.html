Detailed Description

Context information.

Besides segmentation, the text context within the original document can be relevant for checking. Capitalization rules apply differently for titles and headings. Orthographic rules might be different for source code snippets within a software manual. For this purpose, Acrolinx uses context information like XML tag names, style names, or font names as additional text attributes. The linguistic resources can be customized to handle the rules differently depending on the submitted context information.

Detailed Description

Abstract event handler class.

This class is used for informational purpose. When using the SDK in silent mode, the event handler can be used to intercept messages that otherwise would be displayed in dialogs or message boxes. Derive your class from AcroSDK::CAcroEventHandler2 and implement all abstract methods.
